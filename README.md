## 📦 Как запустить проект (Инструкция для команды)
1. Склонируйте репозиторий:
   `git clone https://github.com`
2. Перейдите в папку проекта:
   `cd название-репозитория`
3. Установите зависимости:
   `flutter pub get`
4. Запустите проект:
   `flutter run`
